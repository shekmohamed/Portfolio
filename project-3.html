Dino Game Bot

Developed an automated bot for the Chrome Dino game using JavaScript. The bot uses image processing techniques to detect obstacles and react accordingly.

Project Overview

This project is an automated bot for the Chrome Dino game using JavaScript. The bot uses image processing techniques to detect obstacles like cacti and birds by analyzing pixel changes in the game screen. When an obstacle is detected, the bot automatically triggers a jump by simulating a spacebar key press, allowing the dino to avoid collisions.

The implementation uses JavaScript, and event listeners to analyze the game area in real time and make quick decisions. This project demonstrates basic AI-driven automation, making the Dino play the game without human input.

Tools Used

JavaScript
setInterval() & setTimeout()
Keyboard Event Listeners
Google Chrome Developer Console
